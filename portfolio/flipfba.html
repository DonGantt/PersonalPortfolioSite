Flip FBA

Flip FBA

Client: Flip FBA

Deployment: Production

Programs: Visual Studio Code, AWS

Languages and Frameworks: Custom CSS3, JavaScript, HTML5, DJango, Python

Live At: https://flipfba.com/

Objective: Rebuild the client site using Adobe XD designs provided by the UX/UI team.

I acted as the Front-End developer on this project creating a new version of the their site from the provided Adobe XD design. I worked directly with the art and design team to create their layouts for multiple view ports and device sizes via a completely custom stylesheet built from scratch with a wide array of customizable options. Eventually we upgraded development to the AWS Cloud9 to start the process from local to staging environments and eventually production environments.
