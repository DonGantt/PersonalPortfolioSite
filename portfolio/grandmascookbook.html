Grandma's Cookbook

Grandma's Cookbook

Client: Personal

Deployment: In Beta

Programs: Visual Studio Code, MongoDB, Cloudinary, Passport, Cyclic

Languages and Frameworks: EJS, Node.JS, Mongoose, JavaScript, HTML5, CSS3, Materialize

Live At: https://grandmascookbook.cyclic.app/

Objective: Create a recipe hosting application where users can share and post their recipes with one another.

I developed this project as a personal one in order to flex my skills and learn what it's like to build a back and front end solely from scratch. I started with gathering the resources needed for development and getting my packages in order. Once that was determined, I developed the backend server and tested out its ports along with hooking it up to MongoDB via Mongoose. Once I had the backend in order, I developed the front-end using EJS as my framework with Materialize as my CSS framework. After develop reached MVP stage, I launched the app on Heroku then eventually moved to Cyclic to provide a better user experience with less server downtime. Currently I'm working with UX/UI artist .
